# MCT
Media Converter Termux
Tools sederhana menggunakan ffmpeg
![alt MCT](https://raw.githubusercontent.com/iqbalfaf/MCT/master/2.png)

Tutor Install
apt update
apt install git
git clone https://github.com/iqbalfaf/MCT
cd MCT/
chmod +x install.sh
./install.sh
